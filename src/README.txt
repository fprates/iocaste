Iocaste Framework

* O que é?
Iocaste é um framework para aplicações web.

Iocaste fornece às aplicações:
- um sistema de mensagens entre aplicações;
- autenticação única para a sessão;
- biblioteca de componentes html;
- rastreamento da sessão.
